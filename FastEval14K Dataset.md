# Details
Referenced in ["Xception: Deep Learning with Depthwise Separable Convolutions"](https://arxiv.org/abs/1610.02357) by Chollet (2017).

## Samples
14,000 densely labeled images.

## Classes
About 6,000 classes with 36.5 labels per image.
